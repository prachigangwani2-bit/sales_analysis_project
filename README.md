# sales_analysis_project
Sales data analysis project using SQL, Excel and Power BI. 

A complete sales data analysis project using SQL, Excel and Power BI built on the ClassicModels sales dataset.

## Tools Used
- MySQL - cleaned and analyzed the data
- Excel - created pivot tables and charts
- Power BI - built an interactive dashboard

## What I Found
- Total revenue is $9.6M
- Classic Cars generates the most revenue
- USA is the biggest market
- November has the most orders
- Euro+ Shopping Channel is the top customer
